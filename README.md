# CUHP CTF Platform

A full-stack Capture The Flag platform for cybersecurity competitions with team-based gameplay, leaderboard ranking, and score progression graphs.

## Project Name
CUHP-CTF-Platform

## Quick Start
1. Install dependencies: npm run install:all
2. Configure env files in backend/.env and frontend/.env
3. Seed data: npm run seed
4. Start dev mode: npm run dev

## Structure
- backend: Node.js, Express, MongoDB APIs
- frontend: React, Tailwind, Recharts UI
- docker: Dockerfiles and compose setup
- docs: platform docs and API docs
